# 4. Objetivos 

**Objetivo Geral:** Desenvolver e implementar um sistema automatizado para a gestão da frequência e do desempenho dos alunos de Jiu-jitsu da academia, integrando o acompanhamento técnico e o controle financeiro. 

Objetivos Específicos: 

* Permitir o registro automatizado da frequência. 

* Monitorar a evolução técnica individual. 

* Otimizar a organização das turmas. 

* Melhorar a transparência do desempenho para o aluno. 

* Implementar módulo de controle financeiro para mensalidades
